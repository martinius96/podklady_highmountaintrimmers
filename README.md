# Podklady highmountaintrimmers.com
Podklady e-shopu na doméne highmountaintrimmers.com, ktorú vlastnila spoločnosť eLOC8 s.r.o. vrátane blogovej sekcie pre účely článku vo verejnom záujme ohľadom podnetu na vykonanie kontroly od SOI a ich reakciou.
Materiály webových stránok boli prebraté zo služby WayBack Machine, ktorá mapuje vizuálnu podobu stránok k určitým dátumom.
